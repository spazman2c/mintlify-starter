# Command Reference

Commands use a consistent shape: direction, operation, tenant/customer identifier, environment, and optional mode such as `delta` or `reset`.

```text
<direction> <operation> <tenant-or-customer> <environment> [mode]
```

## Direction conventions

| Prefix | Meaning |
| --- | --- |
| `from-*` | Partner system to IQ Reseller, usually inbound orders, returns, cancellations, or local replay. |
| `to-*` | IQ Reseller to partner system, usually catalog, shipments, invoices, acknowledgements, CRM records, or updates. |
| `bestbuy service` | Service-style Symphony/Best Buy workflow command. |

## SPS Commerce

```bash
from-sps so osi production
to-sps shipment osi production delta
to-sps invoice osi production delta
from-sps solocal osi production delta
to-sps resendasn osi production delta
to-sps resendinvoice osi production delta
to-sps resendsoupdate osi production delta
```
