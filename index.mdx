# IQ Connect Knowledge Base

This Mintlify site turns the raw KT folder into a navigable documentation system. Start with the operational runbooks when you need to support a customer flow, use the integration pages for partner-specific behavior, and use the recordings only when you need original meeting context.

## What is covered

- **27 recording assets** with transcript pages for the KT sessions.
- **131 developer reference files** covering IQConnect, IQR, BrokerBin, Web API, and related architecture notes.
- **82 integration files** across Channable, HubSpot, SPS Commerce, and Best Buy Symphony.
- **17769 source archive entries** inventoried from `iqconnect-master.zip`.

## Recommended path

1. Read [System Map](/start-here/system-map) for the high-level architecture.
2. Use [Command Reference](/operations/command-reference) for production and support commands.
3. Jump to a specific integration page when investigating a customer flow.
4. Search the docs for command names, table names, file names, partner names, or transcript phrases.
